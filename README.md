
# Clipmage

Clipmage is a Windows application that lets you manage the screenshots you take, easier than ever.


## Installation

Download the latest release from the [Releases Page](https://github.com/xanndevs/clipmage/releases/)

#### For Standalone
**1.** Double-click and run the Clipmage.exe

**2.** Take a screenshot

**3.** Drag and drop your screenshot preview onto a supported Drag&Drop field of your choice

#### For Zipped Folder
**1.** Extract and open the folder
 
**2.** Double-click and run the Clipmage.exe

**3.** Take a screenshot

**4.** Drag and drop your screenshot preview onto a supported Drag&Drop field of your choice

## Features

- Draggable preview window
- Drag&Drop feature (keep your mouse still for 100ms while dragging)
- Edit with Snipping Tool
- Pin the preview (Makes it stay on the screen instead of disappearing after a setr amount of seconds)
